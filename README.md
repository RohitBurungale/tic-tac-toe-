Tic Tac Toe Rules

Board
The game is played on a 3×3 grid.

Players
One player uses X, the other uses O.

Turns
Players take turns placing their mark in an empty square.

Goal
The first player to get three of their marks in a row wins.
This can be:

Horizontal

Vertical

Diagonal

Winning
If a player makes a line of 3 X’s or 3 O’s, they win the game.

Draw
If all 9 squares are filled and no one has 3 in a row, the game ends in a draw.

Invalid Move
You can’t place your mark on a square that is already taken.
